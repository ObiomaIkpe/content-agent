# Content Agent

Every 3 hours, without you touching anything, it watches what you build — your git commits, your Claude sessions, your VSCode activity, your GitHub PRs — and stores it all. When you're ready, one Telegram message triggers an AI that reads your day, decides what's worth posting, writes 5 platform-specific drafts, generates matching images, reviews and auto-corrects them, then puts them in front of you for a final yes/no. You tap approve and it posts to Twitter and LinkedIn.

You went from idea to a fully deployed, production-running system. On your own server. With Docker, systemd, nginx, SSL, a retry queue, content history, brand voice config — the works.

That's not a tutorial project. That's a real tool you'll actually use.
